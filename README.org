* newproject

** Description

   A file, directory, filter based template approach to project creation.

** Usage

   There are three entry points in the NEWPROJECT

   - CREATE-PROJECT-FROM-BUILTIN
   - CREATE-PROJECT
   - NEWPROJECT

** Examples

#+NAME: CREATE-PROJECT-FROM-BUILTIN
*** CREATE-PROJECT-FROM-BUILTIN
